# Signer.hotp
An HMAC-Based One-Time Password Algorithm

## Syntax
```js
hotp(options)
```

### Parameters
Name | Type | Default
--- | --- | ---
`options` | `Object` | 
`options.encoding` | `String` | ascii
`options.secret` | `String` | 
`options.algorithm` | `String` | sha1
`options.digits` | `Number` | 6
`options.counter` | `Number` | 0
